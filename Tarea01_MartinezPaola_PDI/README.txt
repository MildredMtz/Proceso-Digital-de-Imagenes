¿Cómo correr el programa?

Dentro del ZIP tendremos los siguientes archivos:
- Imagenfiltro.py
- README.txt
- imagen.jpg

Correremos el programa "Imagenfiltro.py", una vez que lo corramos mostrará el primer
filtro aplicado a "imagen.jpg", tenemos que cerrar esa pestaña (imagen editada) y una 
vez cerrada se cargarán los siguientes dos filtros. Cerramos de igual manera las otras
dos ventanas y el programa guardará en automático las imágenees con los filtros aplicados.

¿Qué hace nuestra práctica?

El propósito de este programa es poder aplicar diferentes tipos de filtros a una imagen,
para esto decidí utilizar python, esto debido a que cuenta con un gran número de 
librerías y en este caso usaré la librería "pillow" la cual es una de las más 
usadas para trabajar con imágenes. 

En cuanto al código podemos ver que para los dos primeros filtros 
(gray_scale y gray_luminance) el código es prácticamente igual donde sólo difiere el 
valor de RGB.
Pero en cuanto al tercer filtro (mica_effect) el código cambia ya que en este caso 
queremos aplicar una transparecia, como nota para este filtro podemos cambiar los 
valores, en este caso se está aplicando un efecto mica azul, pero podemos poner cualquier 
otro modificando la línea señalada en el código.
