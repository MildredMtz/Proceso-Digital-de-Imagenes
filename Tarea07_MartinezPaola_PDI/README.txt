¿CÓMO CORRER EL PROGRAMA?

Lenguaje de programación utilizado: Python 

Dentro del ZIP tendremos los siguientes archivos:
- Erosion.py
- README.txt
- image.png

Nos ubicamos en el archivo "Erosion.py" y corremos el programa. Si lo 
corremos mediante Visual Studio Code simplemente le damos en "run", si lo corremos 
mediante la terminal hacemos lo siguiente:

- Navegamos al directorio donde se encuentra nuestro archivo.
- Una vez que nos encontremos en el direcctorio correcto ejecutamos el archivo con el 
siguiete comando:
      pyhon nombreDelPrograma.py
      python Erosion.py

Despues de hacer esto nos abrirá la interfaz de usuario donde tendremos tres botones
- Cargar Imagen
- Aplicar Erosión Mínima
- Aplicar Erosión Máxima

Primero seleccionaremos la imagen, dentro del zip hay una imágen llamada "image.png", 
una vez que esté abierta la imagen le aplicamos cualqueira de los dos filtros, notemos 
que si seguimos presionando uno de los botones podemos intensificar el filtro.


¿QUÉ HACE NUESTRA PRÁCTICA?
El propósito de este programa es poder aplicar el filtro de erosión máximo (para poder 
hacer las líneas de las letras más gruesas o rellenar huecos) y el filtro de erosión
mínimo (para reducir detalles, hacer las líneas más delgadas o eliminar ruido).
El objetivo de este filtro es ayudar al usuario a hacer más legible en este caso 
archivos o documentos.
