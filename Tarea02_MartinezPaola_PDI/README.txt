¿Cómo correr el programa?

Lenguaje de programación utilizado: Python 

Dentro del ZIP tendremos los siguientes archivos:
- Filtros.py
- README.txt
- imagen.jpg

Nos ubicaremos en el archivo "Filtros.py", corremos el programa y nos arrojará 
la interfaz gráfica, esta interfaz contará con diferentes opciones.
Empezaremos por seleccionar el botón que dice "Cargar Imagen" y seleccionaremos 
la imágen que se encuentra en nuestra  carpeta. Una vez seleccionada le 
podremos aplicar  cualquiera de los filtros que se muestran. 


¿Qué hace nuestra práctica?

El propósito de este programa es poder aplicar diferentes filtros a una imágen,
en total el programa tiene 8 filtros, esto debido a que se pusieron los tres 
de la actividad pasada. A comparación de la otra tarea esta si cuenta con una interfaz
gráfica para que el programa sea más amigable con el usuario.

Para la interfaz se usó tkinter y cv2, de igual manera para los filtros de esta tarea 
se usaron distintas librerías para su ejecución correcta. 
Todo el código se encuentra documentado en caso que se quiera revisar o modificar algo.
