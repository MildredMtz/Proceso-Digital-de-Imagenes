¿CÓMO CORRER EL PROGRAMA?

Lenguaje de programación utilizado: Python 

Dentro del ZIP tendremos los siguientes archivos:
- Escalar.py
- README.txt
- imagen.jpg

Nos ubicamos en el archivo "Escalar.py" y corremos el programa. Si lo 
corremos mediante Visual Studio Code simplemente le damos en "run", si lo corremos 
mediante la terminal hacemos lo siguiente:

- Navegamos desde la terminal al directorio donde se encuentra nuestro archivo.
- Una vez que nos encontremos en el directorio correcto ejecutamos el archivo con el 
siguiete comando:
      pyhon nombreDelPrograma.py
      python Escalar.py

Despues de hacer esto nos abrirá la interfaz de usuario donde tendremos dos botones:
- Cargar Imagen:Sirve para cargar la imagen fuente
- Medida Escalar: Nos abrirá una sub-interfaz para introducir la escala 
- Aplicar Filtro: Definir la escala, esta puede ser de 0.1 a 2.0

Primero seleccionaremos la imagen, dentro del zip hay una imágen llamada "imagen.jpg", 
una vez que esté abierta la imagen seleccionaremos el botón "Medida Escalar", se abrirá 
un nuevo recuadro donde tendremos que seleccionar la escala con un formato de 0.1 a 2.0. 
Una vez definido presionamos "Aplicar Filtro", esperamos unos momentos y tendremos 
nuestra imágen final.


¿QUÉ HACE NUESTRA PRÁCTICA?
El propósito de este programa es poder aplicar una nueva escala a una imagen fuente, 
para ello se utilizaron difentes bibliotecas de python como tkinter, numpy y cv2.
Debido a que cuenta con una interfaz gráfica es más amable con el usuario.
