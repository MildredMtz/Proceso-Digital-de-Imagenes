¿CÓMO CORRER EL PROGRAMA?

Lenguaje de programación utilizado: Python 

Dentro del ZIP tendremos los siguientes archivos:
- ImagenLetras.py
- README.txt
- image.jpg

Nos ubicamos en el archivo "ImagenLetras.py" y corremos el programa. Si lo 
corremos mediante Visual Studio Code simplemente le damos en "run", si lo corremos 
mediante la terminal hacemos lo siguiente:

- Navegamos desde la terminal al directorio donde se encuentra nuestro archivo.
- Una vez que nos encontremos en el directorio correcto ejecutamos el archivo con el 
siguiete comando:
      pyhon nombreDelPrograma.py
      python ImagenLetras.py

Despues de hacer esto nos abrirá la interfaz de usuario donde tendremos tres botones:
- Cargar Imagen:Sirve para cargar la imagen fuente
- Aplicar Mosaico de Letras: Para aplicar el filtro 
- Letra: Podemos definir la letra que queremos aplicar 
- Tamaño región: Tamaño que tomará cada letra dentro de la imágen

Primero seleccionaremos la imagen, dentro del zip hay una imágen llamada "image.jpg", 
una vez que esté abierta la imagen tendremos valores por defecto en "Letra" y "Tamaño
región", estos valores pueden ser cambiados, tome en cuenta que mientras sea menor el 
tamaño región tendremos un mejor resultado.
Una vez definidos los valores anteriores damos click en el botón "Aplicar Mosaico de 
Letras", finalmente nos dará la imágen final con el filtro aplicado. 


¿QUÉ HACE NUESTRA PRÁCTICA?
El propósito de este programa es poder aplicar un filtro a una imágen fuente, de modo 
que la imágen fuente se dividirá en pequeños bloques y cada uno de estos bloques se 
va a sustituir por una letra definida en el programa.
